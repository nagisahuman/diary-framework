What is this?

Diary Framework is a framework that is supposed to be basic. It works well for basic school projects.

What features does it have?

Diary Framework features a rather simplistic interface which is easy to edit. It also features all of these things:

  • A nice .vid tag which centers a video! Or anything else, for that matter.
  • An okay navbar at the top.
  • A .background-needed class which puts a nice background on things. For example, the background behind all of this text!
  • Less than one hundred lines of CSS
